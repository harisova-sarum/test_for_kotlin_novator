Инструкция к решению задачи «Пуск ракеты»


Описание решения:
Для нахождения оптимальной точки запуска выбран метод полного перебора по сетке координат от 0 до 99.


Как скомпилировать:
Если у вас установлен GCC (MinGW):
bash
gcc nuke.c -o nuke.exe


Как запустить:
Программа принимает два аргумента: имя файла с координатами и радиус поражения.
bash
.\nuke.exe coords.txt 10


Если проблемы с кодировкой:
chcp 65001 


Формат входных данных (coords.txt):
Каждая строка должна содержать координаты X и Y, разделенные запятой (без пробелов):
text
10,10
12,12
50,50


Выходные данные:
Программа выведет координаты точки, обеспечивающей максимальный урон, и количество пораженных целей.
